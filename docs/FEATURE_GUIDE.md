# 功能指南

## 1. 系统菜单

系统启动后显示主菜单：

```
***********************************
      欢迎使用导游路线规划系统

请选择你的操作
**********************************

1 重新建立景点图
2 查看景点图
3 生成导游路线
4 查看导游路线中的路径
5 导出导游路线
6 查询两点间最短路径
7 管理景点和道路
8 查看访问统计
9 备份和恢复数据
10 修改密码
11 查看帮助和审计日志
0 退出系统
```

## 2. 功能说明

### 2.1 用户管理

| 功能 | 说明 | 权限 |
|---|---|---|
| 注册 | 创建新用户账号 | 所有用户 |
| 登录 | 使用账号密码登录 | 所有用户 |
| 修改密码 | 修改当前用户密码 | 所有用户 |

### 2.2 景点管理

| 功能 | 说明 | 权限 |
|---|---|---|
| 添加景点 | 添加新的景点 | 管理员 |
| 修改景点 | 修改景点名称、描述、门票、开放时间 | 管理员 |
| 删除景点 | 删除景点及关联道路 | 管理员 |
| 查看景点信息 | 查看景点详细信息 | 所有用户 |

### 2.3 道路管理

| 功能 | 说明 | 权限 |
|---|---|---|
| 添加道路 | 添加两个景点之间的道路 | 管理员 |
| 修改道路 | 修改道路长度 | 管理员 |
| 删除道路 | 删除两个景点之间的道路 | 管理员 |

### 2.4 路径规划

| 功能 | 说明 | 算法 |
|---|---|---|
| 最短路径查询 | 查询两点间最短路径 | Floyd-Warshall |
| 导游路线生成 | 生成完整的导游路线 | DFS |
| 环路检测 | 检测导游路线中的环路 | - |
| 导游路线导出 | 将导游路线导出到文件 | - |

### 2.5 数据备份

| 功能 | 说明 |
|---|---|
| 创建备份 | 创建当前数据的备份 |
| 恢复备份 | 从备份恢复数据 |
| 删除备份 | 删除指定备份 |
| 列出备份 | 列出所有备份 |

## 3. 使用示例

### 3.1 查询最短路径

```
请输入要查询的两个景点名称：
A
C
<A>到<C>的距离是：12.000000米。
<A>到<C>的路径是：
A->B->C
```

### 3.2 修改景点信息

```
请输入要修改的景点名称：
A
请输入景点描述（回车跳过）：
美丽的公园
请输入门票价格（回车跳过）：
50.0
请输入开放时间（回车跳过）：
9:00-17:00
景点信息更新成功。
```

## 4. 注意事项

- 管理员账号：admin/admin
- 删除操作需要输入 "yes" 确认
- 密码最小长度为 6 位
- 备份文件保存在 backup/ 目录下
